> Reviews and ratings for neighborhoods across the country.

## Usage

> Plug in a district, town, city, or zip code and view statistics and community reviews about the area.

  - React/React Native
  - Redux
  - Node v6.9.4
  - Express
  - MongoDB v.3.4.1
  - Mongoose
